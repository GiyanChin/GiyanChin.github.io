Hexo命令
7月 21, 2025

参考：Commands

命令 描述
hexo init [folder] 初始化网站
hexo new [layout] 新建文章
hexo publish [layout] 发布草稿
hexo generate 生成静态文件
hexo server 启动本地服务器
hexo deploy 远程部署
hexo clean 清理数据库和静态文件
hexo list 列出站点信息
hexo version 版本信息
